How to create your own hexo blog -- (learning note)

This is a tutorial for those who what to create their own Blog using Hexo based on Mac OS from scratch.
There are 4 parts including installing, creating hexo config, deploying to remote, and changing themes.
Without further do, let’s start!!!

Quick Start

Installing

  • Before installing Hexo, we should install Node.js first.

  • To check whether you have installed Node.js properly, open Terminal and input following code.

    You need to check npm also.

  • Install Hexo config.

    Then checking whether hexo was installed properly.

Until now, we have finished the first step: installing.

Creating blog config

  • Build a blog folder. Note: you can delete the folder directly if you don’t want the blog any more.

  • Initialize

    Until now, the hexo config has done, we can open the local address to go through it.

  • now, let’s start a new post.

    you can edit the post using Visual Studio Code under the folder ‘source’ of blog, or using vim under terminal as follows.

Deploying to Remote

  • Deploy to github: create a new repository on github. NOTE: the name of the ropos must be same with your username
  • Installing a config under the blog folder.
  • Setup ‘_config.yml’ file. Scrolling to the bottom to find ‘deployment’, and add 3 lines as following:

    type: git
    repo: ’TheNewCreatedGitRepo’
    branch: master

***Note: there should keep one space after : ***
Then, press ’esc’ and input :wq for save and quit vim.

  • Deploy to remote that is github.

Changing the theme

  • Download the theme from github, or search online and install it as their instructions.
  • Application the theme modify ‘landscape’ to ‘yilia’ at the theme of ‘extensions‘
  • Generate, start, and deploy it.

Finally, a personal blog website was done. I will go further into the themes configuring, including Logo, paging, navigation bar and so on.
